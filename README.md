# Exterior Auto Solution Website

Static site for **Exterior Auto Solution** — sparepart mobil bekas & copotan original.

## Quick Start

1. Upload all files to your GitHub repo root (or the `docs/` folder if you prefer Pages from `/docs`).
2. Enable GitHub Pages: **Settings → Pages → Deploy from Branch → `main` → `/root`**.
3. Access: `https://<username>.github.io/<repo>/` or custom domain later.

## Edit Produk

- Data produk ada di `products.json`.
- Foto produk bisa taruh di `assets/img/` lalu ubah path-nya.
- Harga gunakan angka (tanpa titik/koma), format Rupiah otomatis di UI.
